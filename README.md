# 声音素材剪辑和音效合成工具

一个功能完整的浏览器端音频处理工具，使用 Web Audio API 构建，支持音频剪辑、音效生成和各种音频效果处理。

## 功能特性

### 音频加载与播放
- 支持拖拽上传音频文件 (WAV, MP3, OGG, FLAC)
- 导出为 WAV 格式
- 实时播放控制（播放/停止/循环）
- 音量调节
- 波形可视化（支持缩放和导航）

### 基础剪辑功能
- **裁剪选区** - 保留选中的音频片段
- **删除选区** - 删除选中的音频片段
- **淡入淡出** - 可调节时长的淡入淡出效果
- **静音** - 将选区设为静音
- **反相** - 反转波形相位
- **标准化** - 调整音频至最佳电平

### 音效生成器
- **噪音类型**
  - 白噪音 (White Noise)
  - 粉噪音 (Pink Noise)
  - 棕噪音 (Brown Noise)

- **振荡器类型**
  - 正弦波 (Sine)
  - 方波 (Square)
  - 锯齿波 (Sawtooth)
  - 三角波 (Triangle)
  - 频率扫描 (Chirp)

- **ADSR 包络控制**
  - Attack (起音)
  - Decay (衰减)
  - Sustain (延音)
  - Release (释音)

### 音频效果处理
- **均衡器 (EQ)** - 5段参数均衡
- **混响 (Reverb)** - 可调房间大小和衰减
- **延迟 (Delay)** - 带反馈的延迟效果
- **压缩器 (Compressor)** - 动态范围压缩
- **失真 (Distortion)** - 波形失真效果

## 快速开始

### 在线使用

由于浏览器安全限制，您需要通过 HTTP 服务器运行此工具。可以使用以下任意方法：

**方法 1: Python**
```bash
python3 -m http.server 8080
```

**方法 2: Node.js**
```bash
npx http-server -p 8080
```

**方法 3: PHP**
```bash
php -S localhost:8080
```

然后在浏览器中打开: `http://localhost:8080`

### 本地文件安装

1. 克隆或下载此项目
2. 启动本地服务器（见上方）
3. 在浏览器中打开对应地址
4. 拖拽音频文件到上传区域即可开始编辑

## 使用指南

### 1. 上传音频
- 拖拽音频文件到上传区域
- 或点击上传区域选择文件
- 支持同时上传多个文件

### 2. 选择音频片段
- 在波形图上点击并拖动鼠标
- 红色区域表示当前选区
- 选区信息显示在波形图下方

### 3. 基础编辑
- 使用剪辑工具栏中的按钮进行编辑
- 设置淡入淡出时长（秒）
- 支持撤销（需手动重新加载原始文件）

### 4. 生成音效
- 在音效生成器中选择类型
- 设置频率和时长参数
- 调整 ADSR 包络参数
- 点击"生成音效"按钮

### 5. 应用效果
- 勾选要启用的效果器
- 调整各效果的参数
- 效果会实时应用到播放中
- 导出时效果会被渲染到音频文件中

### 6. 导出
- 点击"导出为WAV"按钮
- 处理后的音频会自动下载
- 导出文件名包含时间戳

## 开发说明

### 模块化设计
代码采用 ES6 模块化结构，便于维护和扩展。

### Web Audio API 关键类
- `AudioContext` - 音频处理环境
- `AudioBuffer` - 音频数据容器
- `AudioBufferSourceNode` - 音频源
- `BiquadFilterNode` - 滤波器/EQ
- `ConvolverNode` - 混响
- `DelayNode` - 延迟效果
- `DynamicsCompressorNode` - 压缩器
- `OfflineAudioContext` - 离线渲染

### 扩展开发
要添加新功能：

1. 在对应的模块文件中添加功能
2. 在 `app.js` 中集成新功能
3. 更新 HTML 和 CSS 如需要
4. 测试所有功能

## 常见问题

**Q: 为什么上传后没有声音？**
A: 确认音频文件格式正确，检查浏览器音量和系统音量设置。

**Q: 导出的文件很大？**
A: WAV 是未压缩格式，文件较大是正常的。如需压缩，可使用其他工具转码为 MP3。

**Q: 能撤销编辑吗？**
A: 当前版本不支持撤销，建议在重要操作前先导出备份。

**Q: 支持多轨编辑吗？**
A: 当前版本是单轨编辑，但可以通过导出/导入实现多轨混音。
